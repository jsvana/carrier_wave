# LICW Student Portal

A student-facing portal for Long Island CW Club members to discover classes, track progress, find forums, and access recordings.

## Documentation

- [Product Requirements Document](docs/prd.md)

## Mockups

Open the HTML files in a browser to view interactive mockups:

- [My Schedule](mockups/index.html) - Home view with personalized schedule
- [My Progress](mockups/progress.html) - Curriculum progress and character tracking
- [Forums](mockups/forums.html) - Weekly forum schedule with topics
- [Recordings](mockups/recordings.html) - Past class and forum recordings
- [Explore Classes](mockups/explore.html) - Browse curriculum and recommendations

## Technical Context

- **Platform:** Laravel web app (existing)
- **Database:** Shared with admin.longislandcwclub.org
- **Authentication:** Callsign-based login
